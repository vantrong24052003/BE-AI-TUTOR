## 📱 Page Structure

### Route Map

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         ROUTE MAP                                            │
└─────────────────────────────────────────────────────────────────────────────┘

  PUBLIC ROUTES (No Auth Required)
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  /                           → Homepage                                  │
  │  /marketplace                → Post listing (grid view)                  │
  │  /marketplace/:id            → Post detail                               │
  │  /auth/login                 → Login page (Google OAuth)                 │
  └─────────────────────────────────────────────────────────────────────────┘

  PROTECTED ROUTES (Auth Required)
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  /profile                    → User profile & settings                   │
  │  /profile/purchases          → Purchase history                          │
  │  /profile/purchases/:id      → Purchase detail + credentials             │
  │  /profile/posts              → My selling posts                          │
  │  /profile/posts/new          → Create new post                           │
  │  /profile/posts/:id/edit     → Edit post                                 │
  │  /profile/game-profiles      → My game profiles                          │
  │  /profile/game-profiles/new  → Create game profile                       │
  │  /wallet                     → Wallet overview                           │
  │  /wallet/recharge            → Recharge balance                          │
  │  /wallet/withdraw            → Withdraw (sellers)                        │
  │  /support                    → Support tickets                           │
  │  /support/:id                → Ticket detail                             │
  └─────────────────────────────────────────────────────────────────────────┘

  ADMIN ROUTES (Admin Role Required)
  ┌─────────────────────────────────────────────────────────────────────────┐
  │  /admin                      → Dashboard                                 │
  │  /admin/posts                → Post management                           │
  │  /admin/posts/pending        → Pending posts for approval               │
  │  /admin/purchases            → All purchases                             │
  │  /admin/tickets              → Support tickets                           │
  │  /admin/accounts             → User management                           │
  │  /admin/logs                 → Activity logs                             │
  │  /admin/wallet               → Wallet management (manual recharge)       │
  └─────────────────────────────────────────────────────────────────────────┘
```

## 🎭 Animation & Interaction Guidelines

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         ANIMATIONS                                           │
└─────────────────────────────────────────────────────────────────────────────┘

  HOVER EFFECTS:
  ├── Cards: translateY(-4px) + box-shadow increase
  ├── Buttons: scale(1.02) + brightness(1.1)
  ├── Links: color transition 200ms
  └── Images: scale(1.05) within container

  PAGE TRANSITIONS:
  ├── Fade in: opacity 0 → 1 (200ms)
  ├── Slide up: translateY(20px) → 0 (300ms)
  └── Stagger children: 50ms delay each

  LOADING STATES:
  ├── Skeleton: shimmer animation (linear-gradient moving)
  ├── Spinner: rotate 360deg (1s linear infinite)
  └── Progress: width transition (300ms ease)

  MICRO-INTERACTIONS:
  ├── Button click: scale(0.98) briefly
  ├── Toast: slide in from right
  ├── Modal: fade + scale from 0.95
  └── Dropdown: fade + translateY(-10px)

  ACCESSIBILITY:
  └── prefers-reduced-motion: disable all animations
```
