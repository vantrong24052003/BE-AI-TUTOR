## PROMPT 1: Landing & Auth (4 screens)

```
Design a STUNNING E-learning platform landing page and auth flows that feels premium and futuristic, NOT generic template.

**BRAND VIBE:**
- Futuristic tech education meets premium luxury
- Think: Apple website meets Notion meets Duolingo
- Dark mode primary with vibrant gradient accents
- Glassmorphism, subtle animations, micro-interactions everywhere

**LANDING PAGE:**
Create a jaw-dropping hero section with:
- Animated gradient mesh background (purple → blue → cyan flowing)
- 3D floating elements: books, graduation caps, code snippets rotating slowly
- Bold typography: "Learn Smarter with AI" - massive, gradient text with glow effect
- Subtle particle effects floating in background
- Hero CTA buttons with glass effect, hover glow animation
- Floating mockup of the app interface with depth/parallax

Features section:
- Cards with glassmorphism effect (blur background, subtle border glow)
- Icons with animated gradient strokes
- Hover: cards lift up with shadow, icon pulses

Popular courses:
- Horizontal scroll with snap
- Cards tilt on hover (3D transform)
- Progress bars with animated gradient fill
- Star ratings with sparkle animation on hover

**LOGIN PAGE:**
- Split screen: Left = animated abstract art (gradient blobs morphing), Right = form
- Form container: Frosted glass card floating
- Inputs: Dark with gradient border on focus, glow effect
- Google button: Sleek outline with hover fill animation
- Background: Same gradient mesh but dimmer

**REGISTER PAGE:**
- Same aesthetic as login
- Password strength meter with animated gradient fill
- Celebration confetti animation on successful registration

**FORGOT PASSWORD:**
- Minimal but elegant
- Animated envelope icon that opens/closes
- Success state: Envelope flies away animation

**COLOR PALETTE:**
- Background: #0A0A0F (near black)
- Card: rgba(255,255,255,0.05) with blur
- Primary gradient: #6366F1 → #8B5CF6 → #EC4899
- Accent: #22D3EE (cyan)
- Success: #10B981
- Text: #FFFFFF, #94A3B8 (muted)

Make it feel like a $100M startup's website!
```

```
Design an IMMERSIVE learning dashboard that feels like a next-gen command center, NOT boring admin panel.

- Dark theme with depth layers (different dark shades)

Welcome section:
- "Good morning, Alex" with time-based greeting
- Animated avatar with ring glow
- Streak flame that flickers with CSS animation

Stats cards - Make them FEEL alive:
- Glass cards with subtle inner glow
- Numbers count up animation on load
- Trend arrows with smooth animation
- Background: Subtle animated gradient mesh per card
- Hover: Card scales up slightly, glow intensifies

Continue learning:
- Horizontal carousel with momentum scroll
- Course cards: Rounded corners, gradient borders
- Progress ring that animates on hover
- "Resume" button pulses subtly

Recommended courses:
- Masonry-style grid for visual interest
- Cards have depth shadows
- Category pills with gradient fills

**COURSE LIST:**
- Filters as floating pills (not boring dropdowns)
- Animated filter transitions
- Course grid: Stagger animation on load
- Each card: Image zoom on hover, overlay with quick actions
- Price tag: Gradient pill with glow
- Level badges: Color-coded with icons

**COURSE DETAIL:**
Hero section:
- Full-width gradient header based on category color
- Floating course thumbnail with shadow

Content:

Make it feel PREMIUM, like you're paying $500/month for this!
```

## PROMPT 3: Learning & Quiz (4 screens)

```
Design a CINEMATIC learning experience that feels immersive and engaging, like Netflix meets Duolingo.

**LEARNING PAGE:**
Video player area:
- Cinema mode: Dark overlay, video takes center stage
- Custom video controls: Sleek, appear on hover, fade out otherwise
- Progress bar: Gradient fill with preview thumbnails on hover
- Speed/quality menus: Glassmorphic dropdowns
- Theater mode toggle with smooth transition

Course sidebar:
- Collapsible with smooth animation
- Current lesson highlighted with gradient border + pulse
- Completed lessons: Green checkmark with pop animation
- Locked lessons: Subtle blur + lock icon
- Module headers: Accordion with rotation arrow animation

Bottom bar:
- Previous/Next buttons with arrow animations
- "Mark Complete" button: Fills with gradient on hover, confetti on click
- Notes panel slides up from bottom

**QUIZ PAGE:**
- Full-screen immersive mode
- Timer: Circular progress ring, turns red when low
- Question counter: "3 of 10" with animated transitions

Question card:
- Large, centered, glassmorphic card
- Question text: Clean typography with good hierarchy
- Options: Cards not radio buttons! Hover lifts them
- Selected: Gradient border, checkmark animates in
- Correct/Wrong: Confetti or shake animation

Progress dots:
- Top of screen, horizontal
- Answered: Filled with gradient
- Current: Pulsing ring
- Unanswered: Subtle gray

**QUIZ RESULT:**
Pass state:
- Explosion of confetti + particles
- Large "CONGRATULATIONS!" with glow text
- Score: Animated circular progress that fills up
- Stats in glass cards with icons
- Badges/achievements unlock animation

Fail state:
- Encouraging message, not harsh
- "You're almost there!" with supportive imagery
- Retry button prominent

**AI CHAT:**
- Split screen or full-screen modal
- Chat bubbles: Different shapes for AI vs User
- AI responses: Typing animation, then fade in
- Code blocks: Syntax highlighted with copy button
- Quick suggestions: Floating pills with hover effects
- Voice input button with sound wave animation
- File attachment: Drag & drop zone with glow border

Make learning feel like PLAYING A GAME!
```

## PROMPT 4: Profile & Additional (4 screens)

```
Design profile and utility pages that feel PERSONAL and DELIGHTFUL, not boring forms.

**MY COURSES:**
- Hero banner with user's learning stats
- Tabs: Animated underline slide
- Course cards: Different states (in-progress, completed, paused)
- Progress visualization: Circular rings OR horizontal bars
- Completed courses: Certificate badge with sparkle
- Empty state: Illustration + encouraging message

**PROFILE PAGE:**
- NOT a boring form!
- Avatar: Large, with gradient border ring, edit button overlay
- Stats row: Courses, Hours learned, Certificates, Streak
- Achievement badges section: Grid of unlocked/locked badges
- Learning activity heatmap (like GitHub contribution graph)
- Edit form: Inline editing with smooth transitions
- Save animation: Checkmark burst

**SETTINGS:**
- Sidebar navigation for setting categories
- Each section has its own visual identity

Security:
- Password strength meter (animated gradient)
- Active sessions: Device icons, map of location
- "Log out all" button with confirmation animation

Notifications:
- Toggle switches: Custom design, not default
- Each toggle has icon and color
- Preview of notification types

Appearance:
- Theme switcher: Visual cards (Light/Dark/System)
- Animated transition between themes
- Accent color picker with gradient options

**CHECKOUT/PAYMENT:**
- Order summary: Sticky glass card
- Course info with thumbnail, animated price breakdown
- Payment methods: Card carousel with selection glow
- Card form: Sleek inputs with validation animations
- Security badges: Animated shields, lock icons
- Success state: Confetti explosion, order confirmation

**EXTRAS:**
- 404 page: Fun illustration, "Lost in space" theme, animated
- Loading states: Branded skeleton screens
- Error states: Friendly illustrations, helpful actions

**ANIMATION GUIDELINES:**
- Page load: Stagger fade-up for elements
- Interactions: 200-300ms ease-out
- Success actions: Confetti, checkmarks, celebrations
- Errors: Gentle shake, red flash
- Transitions: Smooth, not jarring

Make every interaction feel REWARDING!
```
